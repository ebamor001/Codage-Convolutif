#Codage Convolutif (MATLAB)

Ce dépôT inclut :

- un encodeur convolutif générique (`cc_encode.m`)
- un décodeur Viterbi en LLR (`viterbi_decode.m`)
- la simulation complète sur canal AWGN
- comparaison de différents codes :
  - variation de la mémoire : (2,3), (5,7), (13,15), (133,171)
  - codes récursifs vs non récursifs : (5,7), (1,5,7), (13,15), (1,13,15)
- tous les scripts de génération de figures
